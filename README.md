# Im Dinh Quoc Bao Khang

Personal full-stack developer portfolio built with Next.js, focused on backend systems, APIs, and production-ready product delivery.

## Live Demo

[https://portfolio.khangyuno.id.vn/](https://portfolio.khangyuno.id.vn/)

## Highlights

- Bilingual portfolio with `EN` / `VI` switching
- Backend-first personal branding and project storytelling
- GSAP-powered motion with optimized language switching
- Radix UI language selector
- SEO metadata for richer link sharing previews
- Scroll-to-top interaction and responsive layout

## Tech Stack

- Next.js 16
- React 19
- TypeScript
- Tailwind CSS v4
- GSAP
- Radix UI
- Biome

## Run Locally

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Available Scripts

```bash
npm run dev
npm run build
npm run start
npm run lint
npm run format
```

## Deployment

This project is deployed on Vercel:

[https://portfolio-khang1z2ts-projects.vercel.app/](https://portfolio-khang1z2ts-projects.vercel.app/)

If you want metadata like canonical URL and share previews to be fully accurate in production, set:

```bash
NEXT_PUBLIC_SITE_URL=https://portfolio-khang1z2ts-projects.vercel.app
```
